# AI 어시스턴트

## AI 어시스턴트는 무엇인가

**AI 어시스턴트**는 사용자의 언어를 이해하고 의도를 파악하여, 질문에 답변하거나 문서를 요약하고 코드를 작성하는 등 다양한 작업을 돕는 지능형 소프트웨어입니다. 대규모 언어 모델(LLM)과 자연어 처리(NLP) 기술을 기반으로 작동하며, 복잡한 명령을 수행하고 사람과 대화하듯 자연스러운 상호작용이 가능한 것이 특징입니다.

현재 시장을 이끌고 있는 대표적인 AI 어시스턴트로는 **ChatGPT**, **Gemini**, **Grok**이 있습니다.
